# JAZZ SENSOR GLOBAL CONFIGURATION
Gradual Adjustments: All software setting changes must be suggested in 5-10% increments.
Extreme Changes: Do not suggest jumps larger than 20% unless the user describes the image as "unusable," "completely black," or "completely white."

- 2x2 Binning: Combines 4 pixels to 1. Increases sensitivity/brightness; decreases resolution. Use only for weak X-ray sources.
- Resend Last Image: Reprocesses previous capture with current settings.
- Processing Chain: Noise Filter > Normalize/CLAHE > Sharpen > Gamma > Out.

# 1. ADAPTIVE NORMALIZATION & S-CURVE LOGIC
- Enable: Activates global histogram stretching.
- Low Percentile (Shadow Removal): Removes the "Dips" or darkest levels.  
    - Logic: Setting value N removes N% of the darkest data levels.  
    - Use: Increase to clear up muddy shadows or "shadow noise."; Default: 1.
- High Percentile (Highlight Removal): Removes the "Peaks" or brightest levels.  
    - Logic: Setting value N removes N% of the brightest data levels.  
    - Use: Increase to reduce "blown-out" or overexposed areas.; Default: 99.
- S-Curve: Controls the inclusion of the "tails" (the data just removed by the percentiles).  
    - Low Scale (0.10 - 0.30): "Hard Clip." Keeps data removed. Best for standard bone/caries checks.  
    - High Scale (0.70 - 0.90): "Soft Clip." Intentionally re-introduces the removed tails. Vital for seeing hairline fractures that are otherwise "clipped".

## THE CLIPPING THRESHOLD (DATA LOSS LIMITS)
- CRITICAL BOUNDARY: High Percentile values > 97% or Low Percentile values > 5%.
- MECHANISM: Setting the High Percentile to 99 removes the top 1% of brightest pixels. If an image is already "bright," increasing this further will permanently "clip" (delete) enamel detail.
- AI SAFETY RULE: If the user complains of "missing detail" or "blown out" areas, the AI MUST check if the High Percentile is too high. You cannot recover data that has been clipped before it reaches the Gamma stage.

# 2. IMAGE ROTATION & INTEGRATION BEHAVIOR
- BEHAVIOR RULE: 
    - TWAIN: Rotation/Flip settings in the Jazz tool WILL affect the final image.
    - FUSE: Rotation/Flip settings in the Jazz tool are DISCARDED; must rotate in host software.

- FUSE INTEGRATIONS (Native - Tool Settings Ignored):
    CDR DICOM, Carestream, Dentrix Ascend, DEXIS, Eaglesoft, Sidexis, Vixwin.

- TWAIN INTEGRATIONS (Tool Settings Accepted):
    XDR, Edge Cloud, Curve Hero, Planmeca Romexis, Oryx, Tigerview, Tracker, iDental, 
    Clio, DTX Studio, SOTA, EzDent-i, Open Dental, Tab32, SOPRO, Mipacs, 
    Denticon XV Capture, Denticon XV Web, CliniView, Dentiray Capture, Imaging XL, 
    Prof. Suni, Xray Vision, SIGMA, PatientGallery, Xelis Dental, Overjet, Aeka, 
    CLASSIC, Archy, Mogo, Acteon, VisionX, OneView, Umbie DentalCare, Harmony.

- DEFAULT FOR UNLISTED SOFTWARE: Any imaging software not named in the FUSE list above
    (including the "OTHER" option) is treated as TWAIN by default — Jazz rotation/flip
    settings should take effect. Confirm by applying a rotation in the Jazz Configurator,
    clicking "Save Settings," and checking that it persists in the final image. If it does
    NOT persist, treat that software as FUSE and use the host software's native rotation controls.

# 3. CLAHE (LOCALIZED CONTRAST)
- Clip Limit: Max contrast enhancement. Range [1.50 - 2.50]; Default: 2.00.
- Num Regions X/Y: Grid tiles for local optimization. Range [4 - 8]; Default: 6x6.

# 4. IMAGE SHARPENING (EDGE ENHANCEMENT)
- Sigma: Blur radius. Range [2.0 - 10.0]; Default: 2.0.
- Weight: Sharpening strength. Range [0.55 - 2.00]; Default: 0.65. (High weights up to 2.00 are used for aggressive edge work, e.g. the Caries recipe.)

# 5. GAMMA & MANUAL ADJUSTMENTS
- Gamma Level: Mid-tone luminance. Range [.35 - .85]; Default: 0.65. DIRECTION: Lower value = BRIGHTER mid-tones; higher value = DARKER mid-tones. The entire range is below 1.0 and the control is capped at 0.85, so Gamma can brighten strongly but can only darken toward neutral — it cannot fully darken a heavily over-penetrated image on its own.
- Manual Contrast: Range [0.00 - 0.10]; Default: 0.00.
- Manual Brightness: Range [-0.10 - 0.10]; Default: 0.00.

# 6. OUTPUT & TWAIN
- Invert: Toggles radiopaque (white bone) vs radiolucent (black bone).
- TWAIN Multiple: Allows multi-frame acquisition.
- TWAIN Wait: Forces trigger wait between frames.
